== Description ==

This module uses Bing Translator [automatic translation](http://edo.webmaster.am/btranslator?xyz=998) service to [translate your web page](http://edo.webmaster.am/btranslator?xyz=998). With 35 available languages your site will be available to 90% of internet users.

Visit [BTranslator Pro](http://edo.webmaster.am/btranslator?xyz=998) for more details and features like `translation editing`, `search engine indexing` and `SEO URLs`.

Please `use` [BTranslator Forum](http://edo.webmaster.am/forum/btranslator/) for your questions and support requests!

* Mouse over effect
* Flags combined in one file to load faster
* Analytics
* Option to open translated page in new window
* Option to translate the page on fly
* Available styles Dropdown/Flags/flags with dropdown
* Valid XHTML
* Option to operate with [Pro version](http://edo.webmaster.am/btranslator?xyz=998) to have SEF URLs, refine/edit translations manually and [index translated pages](http://edo.webmaster.am/btranslator?xyz=998) in search engines

== Installation ==

1. Upload `btranslator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can add a widget to your website or use [btranslator] syntax inside pages where you want it to appear.

== Configuration ==

To configure the widget go to Settings -> btranslator

== Frequently Asked Questions ==

= It doesn't work, what to do? =
Please check the [Troubleshooting](http://edo.webmaster.am/forum/btranslator/troubleshooting-btranslator-free-t370.html) and feel free to create a new topic if you cannot find your case.

= Where I can see analytics data? =
You need to login to your Google Analytics account -> Content -> Event Tracking. The event name will be btranslator and you will see event categories for each language code. If you want to see French language usage you can click on fr and you will see which pages are translated to French by your visitors.

= What is the quality of translation? =
It's a Bing translation, hence Bing quality.

= Which languages are supported? =
Here is the list: Arabic, Bulgarian, Chinese Simplified, Chinese Traditional, Czech, Danish, Dutch, English, Estonian, Finnish, French, German, Greek, Haitian Creole, Hebrew, Hungarian, Indonesian, Italian, Japanese, Korean, Latvian, Lithuanian, Norwegian, Polish, Portuguese, Romanian, Russian, Slovak, Slovenian, Spanish, Swedish, Thai, Turkish, Ukrainian, Vietnamese

= Can I exclude some parts from being translated? =
Yes, you need to wrap the text you don't want to be translated with &lt;span class=&quot;notranslate&quot;&gt;&lt;/span&gt;.
